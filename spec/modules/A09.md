## Goal
Pause / unpause UI in agent shell. If `campaign.pause_codes_required` is FORCE, agent must pick a code; if OPTIONAL, free-form text allowed; if OFF, simple toggle.

## Dependencies
- A01, A03, F02 (pause_codes table)

## Public interface
- `<PauseButton />` toggles READY ↔ PAUSED
- `<PauseCodeMenu />` dropdown when entering PAUSED
- `useAgentState()` hook returns current state + duration
