## Goal
Public REST API for external systems (CRMs, lead vendors, custom apps): `add_lead`, `external_dial` (click-to-call), `update_lead`, `get_lead_status`, `list_dispositions`. API-key authenticated.

## Dependencies
- D01, T04, A04

## Public interface
**REST under `/api/external/v1`:**
- `POST /api/external/v1/leads` — `{ list_id, phone, fields, vendor_lead_code? }` → `{ lead_id }`
- `PATCH /api/external/v1/leads/:id`
- `GET /api/external/v1/leads/:id`
- `POST /api/external/v1/dial` — `{ agent_user_id, phone, lead_id? }` → `{ call_uuid }`
- `GET /api/external/v1/calls/:uuid`
- `GET /api/external/v1/dispositions?campaign_id=`

**Auth:** `Authorization: Bearer <api_key>` per tenant; per-key rate limit + scopes.
