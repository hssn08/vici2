## Goal
The complete "click Dial → call rings → connects to agent's conf" flow for manual mode. Includes a dial dialog, lead-lookup-by-phone, alt-number toggle, dial preview, in-flight cancel.

## Dependencies
- A02 (audio), A03 (state push), T04 (originate), D01 (lead lookup), D03 (TZ gate), D05 (DNC gate), D04 (statuses for hangup mapping)

## Blocks (downstream)
- A05 (live call panel reuses the same call_uuid handle)
- N01 (external click-to-dial reuses same path)

## Public interface
**REST:**
- `POST /api/agent/manual_dial` — `{ phone_e164, lead_id?, alt_dial?, override_dnc?: false }` → `{ call_uuid }` or compliance error
- `POST /api/agent/cancel_dial` — `{ call_uuid }` cancels before bridge

**UI (in agent shell):**
- "Dial Next Lead" button (auto-pulls next from list — Phase 1 = first un-called lead by list_id, by lead_id ASC)
- "Manual Dial" button → modal with phone input + lead lookup
- Live status: ringing → answered → bridged → wrap

**Pseudocode — handler:**
```typescript
async function manualDial(req) {
  const agent = req.user;
  if (agent.status !== 'READY') throw new AppError('AGENT_NOT_READY', 409);
  const phone = normalizeE164(req.body.phone_e164);
  // compliance
  const tz = await tz.resolvePhoneTz(phone);
  const campaign = await getCampaign(agent.currentCampaignId);
  if (!tz.valid) {
    // optionally allow with default tz
  } else if (!isInWindow(phone, campaign.callTimes.start, campaign.callTimes.end, undefined, campaign.callTimes.stateOverrides)) {
    throw new AppError('OUTSIDE_CALL_WINDOW', 403);
  }
  const dnc = await dnc.isDnc(phone, { useFederal: campaign.useFederalDnc, useState: true, useInternal: true, campaignId: campaign.id });
  if (dnc.dnc) throw new AppError('LEAD_DNC', 403, { sources: dnc.sources });
  // lead resolution
  let lead = req.body.lead_id ? await leads.get(req.body.lead_id) : await leads.lookupByPhone(phone);
  if (!lead) lead = await leads.create({ phone_e164: phone, list_id: campaign.defaultListId, status: 'NEW' });
  // originate
  const callUuid = await dialer.originate({
    tenantId: agent.tenantId,
    campaignId: campaign.id,
    leadId: lead.id,
    phoneE164: phone,
    carrierId: campaign.callerIdCarrierId,
    callerID: campaign.callerIdOverride ?? defaultCidFor(campaign),
    timeoutSec: campaign.dialTimeoutSec,
    assignTo: { userId: agent.id },
    recording: campaign.recordingMode,
  });
  await audit({ event: 'manual_dial', user_id: agent.id, lead_id: lead.id, call_uuid: callUuid });
  return { call_uuid: callUuid };
}
```

## Verification phase
1. Click "Manual Dial" → modal; type phone; click Dial; UI shows "Ringing"; customer SIPp answers; UI flips to "On Call"; audio bidirectional.
2. Phone in California at 3 AM Pacific → "Outside call window" modal; no call placed.
3. Phone on internal DNC → "DNC" modal; no call.
4. Lead not in DB → auto-created.
5. Cancel before answer → uuid_kill issued; call_log row marked CANCELED.
6. Two manual dials in rapid succession → second blocked while first is in_progress.
