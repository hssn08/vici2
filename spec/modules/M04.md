## Goal
Admin UI to add/edit/delete SIP carriers (Twilio, RingCentral, BYOC), assign DIDs to inbound destinations (in-group, IVR, agent, voicemail), see live REGED status.

## Dependencies
- T02 (carrier API), F02 (`carriers`, `did_numbers`), I01/I03 (route targets — Phase 3 fills in)

## Blocks
- I02 (DID inbound routing)

## Public interface
**Pages:** `(admin)/carriers`, `(admin)/carriers/[id]`, `(admin)/dids`.

## Research phase
- Per-`kind` form variants; expose only relevant fields per Twilio/RingCentral/BYOC.
