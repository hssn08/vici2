## Goal
Three core reports for Phase 1:
1. **Call Summary** — calls per campaign per day; status breakdown; total/avg talk time.
2. **Agent Productivity** — per agent: calls handled, talk time, wrap time, pause time, dispositions breakdown.
3. **Drop % (TCPA)** — per campaign per 30-day window; rolling daily series; export CSV for compliance audit.
