**Pseudocode — 3-way (DIAL_WITH_CUSTOMER):**
```typescript
async function threeWay(opts) {
  const conf = `conference_${agent.id}`;
  const cid = pickCid(campaign);
  const phone = normalizeE164(opts.target);
  // compliance check on third party (TZ + DNC)
  await assertCompliance(phone, campaign);
  // originate 3rd into the same conf (will join customer + agent)
  await esl.api(`bgapi originate {origination_caller_id_number=${cid},originate_timeout=30,vici2_role=third_party}sofia/gateway/${carrier}/${phone} 'conference:${conf}+flags{join-only}' inline`);
  // event handler will register the new member
}
```

## Verification phase
For each of the 6 kinds, verify with SIPp + 2 fake clients:
1. **Blind**: customer transferred to a number; agent stays in conf for next call.
2. **VM-drop**: customer hears vmdrop audio + hangs up; lead.status='AVMA' (or campaign-defined).
3. **Closer (consultative=NO)**: customer pushed into in-group queue; agent leaves call (Phase 3 once I01 ready).
4. **3-way (DIAL_WITH_CUSTOMER)**: 3rd party rings; on answer joins; all three on the line.
5. **Leave 3-way**: agent drops; customer + 3rd remain bridged; recording continues.
6. **Park + dial**: customer to park MoH; agent dials 3rd alone; on confirm, customer moves to either back-with-agent or closer's conf.
7. **Hangup 3rd only**: drops 3rd; agent + customer remain.
8. **Hangup both**: customer + 3rd both dropped; agent dispo screen.

## Acceptance criteria
- [ ] All 6 modes work; sequence verified by recorded audio + log inspection.
- [ ] Compliance check on 3rd-party number (TZ + DNC).
- [ ] Recording follows transfers (recording_follow_transfer=true).
- [ ] On 3rd-party no-answer, agent gets a clear error; customer remains.
- [ ] Cannot leave-3way if no 3rd party in conf.
- [ ] Audit log per transfer attempt + outcome.

## Risks
- **3rd-party doesn't answer.** Originate timeout fires; customer sits in conf alone with MoH (we set MoH on the customer leg during 3rd ring).
- **Customer hangs up during 3-way setup.** Cleanup logic handles partial state.
- **Park lifetime.** Stale parked customer = bad UX. Add a 10-min park timeout that hangs up + re-dials option later.
