## Goal
Search recordings; stream via signed URL or proxy; in-browser audio player with seek + waveform.

## Dependencies
- R02 (storage_url)
- M01 (admin shell)

## Blocks
- S04 (supervisor recording browser)

## Public interface
**REST:**
- `GET /api/admin/recordings?leadId=&campaignId=&agentId=&from=&to=&page=`
- `GET /api/admin/recordings/:id/url` — returns signed URL (5-min TTL)

**UI:** `(admin)/recordings` page with filters + table + inline player (HTML5 `<audio>` with custom waveform via wavesurfer.js).

## Research phase
- S3 presigned URL format (AWS SDK v3).
- `wavesurfer.js` for waveform.
