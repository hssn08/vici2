- D01, D02, M01, F02 (lists, leads)

**UI pages:** `(admin)/lists`, `(admin)/lists/[id]`, `(admin)/leads`, `(admin)/leads/[id]`.

## Verification phase
1. Create list; assign to campaign; visible.
2. Upload 5k-row CSV; column mapping prompt; commit; rows appear; status counts update.
3. Browse leads with filters (list, status, search by phone/name); pagination works.
4. Open lead detail; edit; save; modify_at updates.
5. Manual create single lead; appears.
6. Bulk status change "DNC" for 100 leads → status updated + DNC entries created.
