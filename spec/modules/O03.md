## Goal
Reproducible load tests: SIPp scenarios for fake carrier (PSTN) + fake browsers (WebRTC); k6 for HTTP/WS; targets capture pre-prod regression baseline.

## Dependencies
- F03, T01, T04

## Public interface
- `scripts/load-test/sipp/{answer,busy,no-answer,reject,3way}.xml`
- `scripts/load-test/k6/{api-load,ws-load,manual-dial-burst}.js`
- `scripts/load-test/run.sh <scenario> <agents> <duration>`
- Reports HTML output with p50/p95/p99/error counts

## Verification phase
1. Run "answer" scenario at 50 CPS for 10 min; FS handles; metrics show stable.
2. Predictive 100-agent scenario; drop% stays under 2%.
3. WebRTC scenario at 200 concurrent; rtpengine offload (X01) keeps quality.
