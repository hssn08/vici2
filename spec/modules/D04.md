## Goal
Manage the per-campaign disposition catalog (NEW, NA, B, A, SALE, NI, DNC, CALLBK, XFER, DROP, …) and the global "system status" mapping (FreeSWITCH hangup_cause → status). Drives the agent disposition picker, hotkeys, status filters, and recycling rules.

## Dependencies
- F02 (`statuses` table)

## Blocks (downstream)
- A06 (dispo picker + hotkeys)
- M07 (admin status admin)
- E01 (hopper filter on dial_status_filter — lookup)
- T04 (hangup_cause mapping)
- D06 (callback recycle uses CALLBK status)

## Public interface
**REST:**
- `GET /api/admin/campaigns/:id/statuses`
- `POST /api/admin/campaigns/:id/statuses`
- `PATCH /api/admin/campaigns/:id/statuses/:code`
- `DELETE /api/admin/campaigns/:id/statuses/:code`
- `GET /api/admin/system-statuses` — global defaults (NEW, NA, B, …)
- `GET /api/admin/hangup-cause-map` — FS hangup_cause → status mapping

**Service:**
```typescript
class StatusService {
  list(campaignId): Promise<Status[]>
  upsert(campaignId, def): Promise<Status>
  resolveFromHangup(campaignId, hangupCause): Promise<string>   // 'NA' | 'B' | etc.
  isSelectable(campaignId, code): Promise<boolean>
  isCallback(campaignId, code): Promise<boolean>
  isDnc(campaignId, code): Promise<boolean>
  hotkeyMap(campaignId): Promise<Record<string, string>>        // { '1': 'SALE', '2': 'NI', ... }
}
```

## Verification phase
1. New campaign auto-inherits default statuses.
2. Add custom "QUOTE_SENT"; appears in agent picker.
3. Set hotkey '1' for SALE, '2' for NI; UI sends them.
4. Map `USER_BUSY` → returns `B`.
5. Cannot delete `NEW` (system-required).

## Test phase

**Run:** `cd api && npm test -- statuses`

## Acceptance criteria
- [ ] Seed populates default statuses for any new campaign.
- [ ] Hangup-cause → status mapping covers all common SIP causes.
- [ ] Per-campaign customization works.
- [ ] System-required statuses can't be deleted.
- [ ] Hotkey conflicts detected (two statuses can't share '1').

## Risks
- **Untranslated hangup causes** drift over time. Default to `NA` for unknowns; log a metric counter so we can extend the mapping.
