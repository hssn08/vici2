## Goal
Agent can serve both outbound campaign AND inbound in-groups. Inbound calls preempt outbound dialing decisions (drop-rate calc accounts for blended).

## Dependencies
- I01, E02, E04

## Public interface
- Per-user-group setting: `closer_ingroups: string[]`
- Picker (E04) considers blended: when computing READY count for outbound, exclude agents with active inbound queue claim.
