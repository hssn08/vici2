## Goal
Embeddable softphone widget that plugs into Salesforce Lightning via Open CTI: click-to-dial from any contact/lead, screen-pop on inbound, dispo writes back to Salesforce via webhook.

## Dependencies
- N01, N02, A02

## Public interface
- Salesforce Open CTI adapter JS at `https://api.example.com/adapter/sf-cti.js`
- Adapter loads our agent UI in iframe inside Salesforce
- postMessage protocol bridges Open CTI ↔ vici2 agent UI

## Verification phase
1. In a Salesforce sandbox, install adapter via Call Center config.
2. Click-to-dial from a contact → vici2 dials → call_log + Salesforce activity created.
3. Inbound call → Salesforce screen-pops the matching contact (via webhook → SF API).
4. Dispo SALE → Salesforce opportunity created (configurable).

## Test phase
**Manual** in SF sandbox; **unit** for postMessage bridge.
