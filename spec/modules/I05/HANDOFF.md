## Mailbox Model

- `voicemail_boxes` table: `id`, `tenant_id`, `ingroup_id`, `user_id`, `did_id`, `greeting_uri`, `max_duration_sec`, `notify_email`, `transcribe`, `active`
- Inbound capture extension naming: `voicemail_{box_id}` in `freeswitch/conf/dialplan/default/75_voicemail_{box_id}.xml`
- Generated by `VoicemailRenderer` (`api/src/services/voicemail/VoicemailRenderer.ts`)
- `api_hangup_hook` is set BEFORE `answer` so partial recordings (caller hung up early) are captured

## VM Drop Audio Format

- Accepted upload formats: WAV (PCM), MP3
- Server transcodes to: WAV 8 kHz, mono, 16-bit PCM (via ffmpeg)
- Duration probed via ffprobe before transcode
- Max duration: 120 seconds
- Max upload size: 10 MB
- Local path pattern: `/var/lib/vici2/vmdrop/{tenant_id}/{asset_id}.wav`
- FreeSWITCH plays via: `uuid_broadcast {uuid} play_and_hangup,{local_path} aleg`
- Unique constraint: `(tenant_id, name)` on voicemail_drop_assets

## AMD Integration

- AMD event stream: `events:vici2.call.amd_detected` (Valkey XREADGROUP)
- Handler: `dialer/internal/picker/amd_handler.go` — `AMDHandler.handle()`, case `"vmdrop"`
- Config plumbing: `CampaignConfig.AMDAction` + `CampaignConfig.VMDropPath` from M02 config snapshot JSON
- Success path: `UUIDBroadcast play_and_hangup,{path} aleg` → increments `vici2_picker_vmdrop_played_total`
- Fallback (no asset): plain `UUIDKill` + `vmdrop_blocked_no_asset` audit event → increments `vici2_picker_vmdrop_fallback_total{reason="no_asset"}`
- Audit stream: `events:vici2.audit.vmdrop`; consumed by `vmdrop-audit-consumer.ts` → `audit_log`

## Key Configuration Fields

- `Campaign.amdAction` = `vmdrop` (enum)
- `Campaign.vmdropAssetId` = FK → `voicemail_drop_assets.id`
- `Campaign.vmdropRequiresConsent` = `true` (default) — triggers Gate 5 TCPA consent check
- `VoicemailBox.notifyEmail` — team email for new-voicemail notification (BullMQ → N02)

## Partial Recording Detection

- `api_hangup_hook` fires on ALL hangup paths (caller, callee, FS timeout)
- `partial` field on voicemails: `true` if `duration_sec < 3` OR `file_path` empty OR explicit `partial=true` flag from hook
- Audit action: `voicemail_partial` vs `voicemail_captured`

## RBAC Verbs

- `vmdrop:read` — super_admin, admin, supervisor (scope: tenant)
- `vmdrop:edit` — super_admin, admin (scope: tenant)
